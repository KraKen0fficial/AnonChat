# AnonChat

Красивый анонимный мессенджер в реальном времени на Node.js + WebSocket.

## Возможности
- Анонимный вход без регистрации.
- Комнаты по коду (можно делиться ссылкой).
- Реальное время (WebSocket).
- Красивый адаптивный интерфейс с glassmorphism, иллюстрацией и панелью SVG-иконок.
- Индикатор подключения и списка участников.
- История сообщений сохраняется на сервере в `data/history.json` (последние 100 сообщений на комнату).

## Локальный запуск
```bash
npm install
npm start
```

Откройте: `http://localhost:3000`

## Деплой (прямая ссылка для всех)
### Вариант 1: Render (рекомендуется)
1. Загрузите репозиторий на GitHub.
2. На https://render.com создайте **Web Service** из этого репозитория.
3. Настройки:
   - Build Command: `npm install`
   - Start Command: `npm start`
4. После деплоя Render выдаст публичную ссылку вида:
   `https://your-app-name.onrender.com`

### Вариант 2: Railway
1. Импортируйте репозиторий с GitHub в Railway.
2. Railway автоматически определит Node.js приложение и выдаст публичный URL.

## Структура
- `server.js` — WebSocket сервер + раздача фронтенда
- `public/` — клиент (HTML/CSS/JS)
